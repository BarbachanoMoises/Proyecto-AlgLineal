El programa se ejecuta haciendo doble click en el ejecutable ElimGauss.exe.
El archivo input1.txt debe encontrarse obligatoriamente en el directorio de ElimGauss,
con el formato adecuado.
El input1.txt deberá tener el siguiente formato:

2 3  # 2 renglones, 3 columnas
4 2 -1 6  # Datos
3 -2 0 4

(Los comentarios se ponen para aclarar, pero el archivo real no debe poseer comentarios).
