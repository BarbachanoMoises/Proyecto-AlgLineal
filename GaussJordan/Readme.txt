El programa se ejecuta haciendo doble click en el ejecutable GaussJordan.exe.
El archivo input2.txt debe encontrarse obligatoriamente en el directorio de GaussJordan,
con el formato adecuado.
El input2.txt deberá tener el siguiente formato:

2   # Número de sistemas
2 3  # 2 renglones, 3 columnas
4 2 -1 6  # Datos
3 -2 0 4
3 3  # Tamaño del 2o sistema
2 3 1 5  # Datos
3 4 -1 3
1 -2 1 7

(Los comentarios se ponen para aclarar, pero el archivo real no debe poseer comentarios).
